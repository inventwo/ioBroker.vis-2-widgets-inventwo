# vis-2-widgets-inventwo — Documentation

This folder contains the user documentation for the **inventwo** widget set for ioBroker VIS 2.

The widgets let you build fully styled dashboards — buttons, sliders, tables, dropdowns, scrolling text and more — directly in the VIS 2 editor, without writing any code.
